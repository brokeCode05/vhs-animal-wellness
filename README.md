# VHS Animal Wellness Center — Static Demo

This is the **GitHub Pages static demo** of the VHS Animal Wellness Center web system.

## What works on this demo
- Public website: Home, Services, About, Contact pages
- All animations, carousels, and UI interactions
- Admin, Clerk, and User dashboard layouts (view only)

## What requires a backend server
- Login / Sign Up (requires PHP + MySQL)
- Appointment booking
- Pet registration
- Profile management
- Email verification

## Full deployment
The complete system (with PHP backend) is designed to run on a PHP 8+ / MySQL server such as:
- InfinityFree, 000webhost, or any cPanel shared host
- A VPS with Apache/Nginx + PHP + MySQL

## Backend files
All PHP files are in the `_backend/` folder for reference.
The production database schema is at `_backend/vet_db.sql`.
